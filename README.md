# 🤖 AI 资讯 Agent

> 输入任意话题，自动搜索全网资讯，AI 深度分析生成结构化报告。

**🔗 在线体验：[https://ed-dddddi-li.github.io/ai-news-agent/](https://ed-dddddi-li.github.io/ai-news-agent/)**

**技术栈**：HTML · CSS · JavaScript · Tavily Search API · DeepSeek API

---

## ✨ 功能

- 🔍 **全网搜索**：接入 Tavily 搜索引擎，自动抓取最新资讯（3-5条来源）
- 🧠 **AI 分析**：DeepSeek 对搜索结果深度整理，生成结构化报告
- 📊 **多种风格**：简洁摘要 / 深度分析 / 要点清单
- 📎 **来源追溯**：每条结论都附带原始来源链接
- 🌐 **支持中英文**输出

---

## 🚀 使用方法

1. 打开[在线地址](https://ed-dddddi-li.github.io/ai-news-agent/)
2. 配置 API Key（需要 DeepSeek + Tavily 各一个）
3. 输入任意话题或点击快捷话题
4. 点击「开始搜索」，等待 Agent 完成四步流程

> 所有 Key 均在本地使用，不会上传至任何服务器
